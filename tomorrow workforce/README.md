# Tomorrow Workforce Website Wireframe

This folder contains the first launch of the Tomorrow Workforce website wireframe.

## Files

- `index.html` - Static homepage wireframe with sections for hero, problems, solutions, process, industries, integration, and contact.
- `styles.css` - Basic responsive styles for the homepage.
- `script.js` - Dynamic section rendering and contact form handling.
- `site.js` - Shared page utilities, mobile navigation, and active navigation highlights.
- `solutions.html`, `how-we-work.html`, `case-studies.html`, `about.html`, `contact.html` - supporting site pages.
